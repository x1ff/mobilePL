## Run a test using
1. Создайте в корене проэкта директорию apk и разместите в ней apk файл
2. Создайте переменную окружения с именем  APK_DIR, укажите в ней абсолютный путь до директории
2. Запустите сервер апиум

```appium```

3. Запустите эмулятор и посмотрете udid

```adb devices```
4. Измените настройки в application.properties

5. Нажимаем треугольник в идее
